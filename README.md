### Open Source Work

- **[StageX](https://stagex.tools)** — FROM scratch toolchains for reproducible containers. All bootstrapped from stage0, no apt/pip/yum.
- **[zeroclaw](https://github.com/zeroclaw-labs/zeroclaw)** — CI/CD supply-chain security: SLSA provenance, SBOM, lockfile integrity, dependency review. Rust toolchain.
- **[Thaleia](https://codeberg.org/ConYel/thaleia)** — Voice AI in Rust. TTS inference with Kokoro-style models, local-first, no cloud dependency.
- **[WIND](https://github.com/ConYel/wind)** — wORKFLOW FOR PiRNAs AnD BEYONd. piRNA analysis pipeline.

### DNS Sinkhole Benchmarks (coming soon)

Comparative performance analysis of Pi-hole, Blocky, AdGuard Home, Ferrous DNS, and Numa DNS — all compiled as FROM scratch static binaries via StageX. 5 implementations × 4 upstream providers × 2 modes = 192 configurations. Memory, latency, throughput.

### Background

PhD in Molecular Oncology (Magna Graecia University). Previously at ONCOLille Research Center (University of Lille) as an MSCA Early Career Researcher, and at the Laboratory of Molecular Medicine and Genomics (University of Salerno). EvoCELL ITN alum.
